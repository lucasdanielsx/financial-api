# Getting Started

### How to run?
You can use one of the follows commands:

`mvn spring-boot:run`

or docker:

`mvn clean install`

`docker build -t financial-api .`

`docker run -p 8080:8080 -t financial-api:latest`

or with docker compose:

`mvn clean install`

`docker-compose up`
